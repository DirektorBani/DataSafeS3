# Настройка внешнего S3

Опциональный шаг мастера настройки или позже через **Admin → Gateway**.

## Сценарии

- Асинхронная репликация bucket на удалённой стороне или AWS S3
- Гибридное хранение (горячее локально + холодное удалённо)
- Цель аварийного восстановления

## Поля мастера

| Поле | Пример |
|------|--------|
| Endpoint | `http://minio:9000` |
| Ключ доступа | ваш ключ |
| Секретный ключ | ваш секрет |
| Бакет | `datasafe-backup` |
| Регион | `us-east-1` |
| Использовать SSL | переключатель |

## Тест подключения

```http
POST /api/v1/setup/s3/test
```
