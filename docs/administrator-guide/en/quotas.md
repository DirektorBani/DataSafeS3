# Quotas

Limit storage usage per user or bucket.

## User quotas

Set `max_size_bytes` and `max_objects` when creating/editing users in **Administration → Users**.

## Bucket quotas

Configure in bucket settings or **Admin → Settings → Buckets**.

## Enforcement

Uploads are rejected when quota is exceeded. Dashboard and usage pages show consumption.
