# DataSafeS3 Mobile Web (Phase 4 preview)

Lightweight PWA for phones — browse buckets, download and upload files. Runnable with Node only (no Flutter/Rust).

```bash
cd clients/mobile-web
npm install
npm run dev
```

Open the dev URL on a phone on the same network (use your PC LAN IP instead of `localhost`).

Production Flutter app: [../mobile/README.md](../mobile/README.md).
