# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in the DataSafeS3 project a harassment-free experience for everyone.

## Standards

Examples of acceptable behavior include using welcoming language, respecting differing viewpoints, and accepting constructive criticism.

Unacceptable behavior includes harassment, trolling, and publishing others' private information without permission.

## Enforcement

Project maintainers may remove or reject contributions that violate this code. Report issues to the repository owner.

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
